Controls

Roles & permissions

Only admins can change settings, run cost sync, and apply write-backs. Non-admin users can view dashboards and insights.

Escalation contacts

Configure who is notified at each escalation level and for which incident types. Editable in Settings.

Approval thresholds

Set thresholds for when an action requires approval (e.g. cost, risk, or policy). Thresholds are configurable per site.

Decision tracking

Each decision has a planned action, applied? flag, expected impact, and review date. Outcome status: Pending / Implemented / Reviewed. Admins can update status and add outcome notes so the system maintains an audit trail.
